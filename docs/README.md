# AE-OS — Autonomous Enterprise Operating System

Documentation set for building a self-evolving, multi-department, revenue-maximizing autonomous business operating system, with human (Chairperson) final veto.

## Read in this order

1. `00_MASTER_PROMPT.md` — root system prompt for the AI CEO orchestrator
2. `01_CLAUDE_PROJECT_INSTRUCTIONS.md` — CLAUDE.md for Claude Code sessions
3. `02_ARCHITECTURE.md` — full layered system architecture
4. `03_AGENT_SPECS.md` — every sub-agent, its job, and its Definition of Done
5. `04_OPERATING_LOOP.md` — the exact daily/event-triggered execution cycle
6. `05_SELF_EVOLUTION.md` — how the system grows agents, cuts cost, and scales over years
7. `06_MEMORY_SYSTEM.md` — shared vector/graph memory schema and access rules
8. `07_TECH_STACK_AND_INFRA.md` — model routing, stack choices, cost discipline
9. `08_GOVERNANCE_AND_SAFETY.md` — approval gates, budgets, kill switch (non-negotiable)
10. `09_TRILLION_INTEGRATION.md` — what's borrowed from Trillion's design and why
11. `10_BUILD_ROADMAP.md` — phased plan from zero to production
12. `11_TOKEN_OPTIMIZATION.md` — system instructions for optimized token usage

Governance (`08`) overrides every other file in case of conflict.
